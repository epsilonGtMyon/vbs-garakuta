# sjis-to-utf8-without-bom

SJISで保存されたテキストファイルをBOMなしのUTF-8で出力する。

## 使い方


|  引数     |  必須  | 説明                                              |
| --------- | ----- |---------------------------------------------------|
|  第1引数  |  必須  | 変換元ファイルのパス                                |
|  第2引数  |  任意  | 変換先ファイルのパス(省略時は元ファイルを書き換える)   |


コマンドラインで使う場合

```
SjisToUtf8WithoutBom.vbs aaa.txt
SjisToUtf8WithoutBom.vbs aaa.txt bbb.txt
```

直接 D&D した場合は元ファイルを書き換えます。
